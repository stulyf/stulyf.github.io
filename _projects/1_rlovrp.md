An online mobile charging vehicle routing and scheduling system that combines heuristic algorithms with DQN reinforcement learning. The system models real-world scenarios with dynamic request points and time-window constraints, using an intelligent policy selector to dispatch mobile charging stations efficiently.

## Key Features

- **Problem Modeling**: Formulates the online vehicle routing problem with dynamic customer requests and time-window constraints
- **Hybrid Algorithm**: Combines traditional heuristic methods with DQN reinforcement learning for online scheduling decisions
- **Intelligent Policy Selector**: Automatically selects the best dispatch strategy based on current scenario conditions
- **Visualization & Evaluation**: Provides route trajectory visualization and performance evaluation tools
